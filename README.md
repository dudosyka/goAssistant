# GoAssistant

# #1 How to install:
    1. Go to project directory
    2. Open prompt
    3. Write down `npm i`

# #2 How to start application:
    1. Go to project directory
    2. Open prompt 
    3. Write down `npm run serve`

# #3 How to build application:
    1. Go to project directory
    2. Open prompt
    3. Write down `npm run build`
